# Proyecto-videojuego-del-gato---tres-en-raya

Considerando el juego clásico del gato. En dicho juego gana quien hace tres marcas de forma: vertical horizontal o diagonal. 
Use una matriz de tres por tres 
Se pueden usar 0 y1 o bien O y X
Primero deberá tirar el humano eligiendo una posición del a1 a 9  [1,2,3    
                                                                                                            4,5,6
                                                                                                            7,8,9]
Luego la computadora. Así hasta que uno de los gane. Al final; Gana el humano, Gana la computadora o Nadie ganó
Considere la siguiente matriz con las siguientes posiciones:
0,0   0,1   0,2  
1.0   1.1   1.2
2.0   2.1   2,2

Después de cada tirada revisar si alguien ganó, sino sigue. si alguien ganó termina el juego. 
Se deberá mostrar los más gráficos posible con Tkinter
